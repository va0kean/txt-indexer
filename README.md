# TXT Indexer



Графическое приложение на Python (Tkinter) для индексации TXT-файлов.



\## Возможности

\- Индексация тысяч `.txt` файлов

\- Извлечение:

&nbsp; - Категория

&nbsp; - Автор

&nbsp; - Название

\- Сортировка по колонкам (А–Я)

\- Открытие файла двойным кликом

\- Кэш индекса в SQLite

\- Инкрементальная переиндексация



\## Запуск

```bash

python indexer\_gui.py
